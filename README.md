📌 Project Highlights
1. Exploratory Data Analysis
Data cleaning and preprocessing
Missing value handling
Survival analysis on Titanic dataset
Visualizations using Seaborn and Matplotlib
2. Simple Linear Regression
Predicting sales using advertising data
Built using Scikit-learn
Evaluated using R², MSE, and RMSE
3. Classification Model
Breast cancer prediction
Logistic Regression and Decision Tree models
Evaluated using Accuracy, Precision, Recall, and F1-score
4. Sentiment Analysis
IMDB movie review classification
Text preprocessing and TF-IDF
Logistic Regression for sentiment prediction
5. Recommendation System
Collaborative Filtering using MovieLens dataset
User-item matrix creation
Cosine similarity for movie recommendations
6. Data Visualization Dashboard
Interactive dashboard using Dash and Plotly
Multiple charts and live filters
Titanic dataset insights visualization
